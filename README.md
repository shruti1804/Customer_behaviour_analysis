# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end Data Analytics workflow — from raw dataset processing to business insights and visualization.

The objective of this project is to analyze customer transactional data, uncover meaningful patterns, and generate actionable insights using Python, SQL, and Power BI.

It highlights practical skills in data cleaning, exploratory data analysis, SQL querying, dashboard development, and business reporting.

---

## Dataset

The dataset consists of customer-level transactional records including:

- Customer ID  
- Gender  
- Age  
- Product Category  
- Item Purchased  
- Purchase Amount  
- Discount Applied  
- Subscription Status  
- Shipping Type  
- Review Rating  
- Previous Purchases  

The data was initially loaded and cleaned using Python before being stored in a relational database for SQL-based analysis.

---

## Tools

- **Python** (Pandas, NumPy, Matplotlib, Seaborn)  
- **PostgreSQL / MySQL / SQL Server**  
- **Power BI**  
- **Jupyter Notebook / Google Colab**  
- **Gamma (for presentation creation)**  

---

## Steps

### 1. Data Loading & Cleaning
- Imported dataset using Pandas  
- Handled missing values  
- Removed duplicates  
- Corrected data types  
- Standardized categorical values  

### 2. Exploratory Data Analysis (EDA)
- Analyzed revenue distribution  
- Compared spending by gender  
- Segmented customers by age groups  
- Studied discount usage patterns  
- Evaluated subscription behavior  
- Identified top-performing products  

### 3. SQL Analysis
- Loaded cleaned dataset into PostgreSQL/MySQL/SQL Server  
- Wrote analytical queries to compute:
  - Revenue by gender  
  - Top products by rating  
  - Customer segmentation  
  - Discount impact analysis  
  - Subscription vs non-subscription revenue  
  - Revenue contribution by age group  

### 4. Dashboard Development
- Connected database to Power BI  
- Built interactive visualizations  
- Designed KPIs and performance metrics  
- Created filters for dynamic data exploration  

### 5. Reporting & Presentation
- Prepared a structured analytical report  
- Designed a professional presentation using Gamma  
- Summarized findings with clear business insights  

---

## Dashboard

The Power BI dashboard includes:

- Total Revenue KPI  
- Revenue by Gender  
- Revenue by Age Group  
- Top 5 Products  
- Subscription vs Non-Subscription Analysis  
- Discount Impact Visualization  
- Customer Segmentation Overview  

The dashboard allows interactive filtering and supports data-driven decision-making.

## Results

- Identified high-revenue customer segments

## How to Run

1. Clone the repository.  
2. Install required Python libraries (Pandas, NumPy, SQLAlchemy, etc.).  
3. Run the Python notebook to clean and prepare the dataset.  
4. Load the cleaned dataset into PostgreSQL/MySQL/SQL Server.  
5. Execute the provided SQL queries.  
6. Open the Power BI (.pbix) file to explore the dashboard.  
7. Review the final report and presentation created in Gamma.
